# Notary Journal Static Pages

Open this folder with Live Server and use these entry files:

- `src/index.html`
- `src/pages/admin.html`
- `src/pages/notary/notary.html`
- `src/pages/user/user.html`

Folder layout:

- `src/css/` shared and page-specific styles
- `src/js/common/` helpers for loading partial HTML
- `src/js/admin/` admin page logic
- `src/js/notary/` notary page logic
- `src/js/user/` user page logic
- `src/pages/admin/partials/` dashboard sections
- `src/pages/notary/partials/` journal list sections
- `src/pages/user/partials/` user flow sections
